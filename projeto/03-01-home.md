# Home (Index)

## Objetivo
- Ser a pagina principal e concentrar informacoes do grupo.

## Escopo
### Inclui
- Hero com nome do grupo e chamada principal
- Secao "O que e NA"
- Secao "Historia do Grupo"
- Secao "Relatorios de Servico"
- Bloco de reunioes (dias/horarios, ID, senha, link Zoom)
- Link externo para Looker Studio

### Nao inclui
- Painel administrativo ou consulta de participantes
- Cadastro manual de presencas

## Arquitetura de Layout
- Single-page com navegacao interna via JS (`navigateTo`)
- Menu desktop + menu mobile
- Secoes com classe `page-section` e visibilidade controlada
